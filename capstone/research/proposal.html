Capstone Project Proposal

The topic I plan to pursue, at least right now, for the Capstone project is an educational, illustrative and interactive game meant for younger children. I'm thinking of having some kind of scavenger hunt or reveal for each part of the seal - either like "cards" that turn over when you click on them (and you can win different things depending on what you clicked on), like some sort of memory game if there's duplicate types of cards that I can make the seal parts into. An illustration that scatters the different parts/items that represent what makes up the seal would for the scavenger hunt, which I think would be really fun for kids as well. I currently do not have plans to work with a partner, but if it comes up I would be happy to work with one. I'm also happy to do it on my own - either one! Some of the features that I would imagine for this project would include zoom in, hover, click features with sound effects and bouncing/slight moving animations on graphics. It would have a cute, simple illustrative style and would guide users through the process easily and give lots of rewards! Possibly a scroll function to go from top to bottom - and now that I'm imagining that maybe some kind of winding road going from top to bottom as some sort of race would be fun too! Or just as a scavenger hunt.

I'm a little concerned about the graphics - I'll need to make sure I don't overdo it and spend too much time on the overall illustrations, as I tend to do that. Going for minimalist and remembering that it doesn't have to be perfect is important. Getting the focus point/animal/person to move and follow the path and react when they find something would be a challenge. I may also choose to not use a path or scrolling, and simply have a landscape scenery that just fits the screen and has points you can click on to find items (with informational pop-ups and rewards). I'll want to implement new JS and possibly json tools/skills as well, which will be a challenge as I'm still understanding them.
