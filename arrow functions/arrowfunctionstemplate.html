Arrow Functions

This function is the inverse of what we learned from the examples, and makes everything lowercase.

Named Function

            STRUCTURE:
            function(parameter){
                Do something();
            }

            EXAMPLE:
            function makeLowerCase(nocaps) {
                return nocaps.toLowerCase();
            }
            console.log(makeLowerCase("LOWERCASE"));

Function Expression

            STRUCTURE:
            const aVariable = function(parameter){
                Do something();
            }

            EXAMPLE:
            const makeLowerCase = function (nocaps) {
                return nocaps.toLowerCase();
            }
            console.log(makeLowerCase("LOWERCASE"));

Arrow Function

            STRUCTURE:
            const aVaraiable = arrow function...{
            }

            EXAMPLE:
            const makeLowerCase = (nocaps) => {
                return nocaps.toLowerCase();
            }
            console.log(makeLowerCase("LOWERCASE"));

            MORE CONCISE EXAMPLE:
            const makeLowerCase = nocaps => nocaps.toLowerCase();
            console.log(makeLowerCase("LOWERCASE"));
